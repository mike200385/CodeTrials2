Detailed Description

GAIEcommerceProduct.h Google Analytics iOS SDK Hit Format Header

Copyright
Copyright 2014 Google Inc. All rights reserved.

Class to construct product related information for a Google Analytics beacon. Use this class to report information about products sold by merchants or impressions of products seen by users. Instances of this class can be associated with both Product Actions and Product Impression Lists.
Typical usage: [tracker set:kGAIScreenName value:"MyScreen"]; GAIDictionaryBuilder *builder = [GAIDictionaryBuilder createScreenView]; GAIEcommerceProduct *product = [[GAIEcommerceProduct alloc] init]; [product setId:""PID-1234""]; [product setName:@"Space Monkeys!"]; [product setPrice:@100]; [product setQuantity:@2]; [builder addProductImpression:product impressionList:"listName"]; [tracker send:[builder build]];
