Detailed Description

GAIProductAction.h Google Analytics iOS SDK Hit Format Header

Copyright
Copyright 2014 Google Inc. All rights reserved.

Class to construct transaction/checkout or other product interaction related information for a Google Analytics hit. Use this class to report information about products sold, viewed or refunded. This class is intended to be used with GAIDictionaryBuilder.
Typical usage: [tracker set:kGAIScreenName value:"MyScreen"]; GAIDictionaryBuilder *builder = [GAIDictionaryBuilder createScreenView]; GAIEcommerceProductAction *action = [[GAIEcommerceProductAction alloc] init]; [action setAction:kGAIPAPurchase]; [action setTransactionId:"TT-1234"]; [action setRevenue:@3.14]; [action setCouponCode:"EXTRA100"]; [builder setProductAction:action]; GAIEcommerceProduct *product = [[GAIEcommerceProduct alloc] init]; [product setId:""PID-1234""]; [product setName:@"Space Monkeys!"]; [product setPrice:@100]; [product setQuantity:@2]; [builder addProduct:product]; [tracker send:[builder build]];
