Detailed Description

Extends UIViewController to generate Google Analytics screenview calls whenever the view appears; this is done by overriding the viewDidAppear: method. The screen name must be set for any tracking calls to be made.

By default, this will use [GAI defaultTracker] for tracking calls, but one can override this by setting the tracker property.
